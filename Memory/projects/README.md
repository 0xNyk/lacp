# Memory / Projects

Purpose
- Durable project-level memory for active and archived initiatives tied to LACP.

What to store
- Goals and success metrics
- Key architecture decisions
- Operating constraints and risk profile
- Current status, blockers, next milestones
- Canonical commands/artifacts for verification

Template
- Project:

- KPI(s):
- Current phase:
- Key decisions:
- Risks:
- Canonical commands:
- Next actions:

Conventions
- Keep one markdown file per project stream.
- Link to relevant docs/runbooks instead of duplicating long content.
- Timestamp major updates.
