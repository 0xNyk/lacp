# Memory

Phase 6 memory scaffold for LACP.

Structure
- `people/` — durable stakeholder/operator notes
- `projects/` — project stream memory and status
- `context/` — environment and operational invariants
- `glossary.md` — shared language and metric definitions

Usage
- Keep entries concise, factual, and easy to verify.
- Prefer links to canonical docs over duplication.
- Update `Heartbeat.md` when major capabilities ship.
